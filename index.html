Learn Git

This is a project to learn git and how to use it.

Getting started with git

  1. git init - Initialized git in current folder
  2. git config --global user.name "Suraj Oliver"
  3. git config --global user.email "suraj.oliver@gmail.com"
  4. git status (Check status of git folder)
  5. git add -A/--all (Add all untracked files to the repo)
  6. git restore filename (Restore files from repo)
  7. git commit -m "first release" (Its a savepoint - Stage to commit)
  8. git log

Git maintains two status for each files (a) untracked - for files in current folder which are not added to repo and (b) tracked - for files added to repo.

Git Branches

  1. git branch branch_name

Uses of branch - Helps create multiple branch to work in a large project. In case of bug fixes, we can create a new branch from the main branch called small-error-fix. We can fix the error in the small-error-fix and then merge it with the main branch. When our dev is finished. we can add the small-error-fix to be dev branch also.
